## Key Observations Summary

1. **Extreme restraint** — Only 2 colors matter: black and the accent. Everything else is white/gray.
2. **Typography-driven hierarchy** — Size and weight do all the work, not color or decoration.
3. **Generous spacing** — Content breathes. Nothing feels cramped.
4. **Consistent patterns** — Every screen follows the same component rules. Zero surprises.
5. **Single-task focus** — Each screen does one thing well. No dashboards, no multi-panel layouts.
6. **Progressive disclosure** — Information is revealed in layers. "Show all" buttons are everywhere.
7. **Trust through transparency** — Helper text explains WHY data is needed. Legal text is visible but not obstructive.
